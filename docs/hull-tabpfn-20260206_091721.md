## GPU 监测规则
- 默认强制 GPU（`-RequireGPU` 默认 `true`）
- 监测指标：GPU 利用率与显存占用（二者同时低于阈值才视为未使用 GPU）
- 可调参数：`-GpuMinUtil`、`-GpuMinMemoryMB`、`-GpuCheckDelaySec`、`-GpuCheckWindowSec`、`-GpuCheckIntervalSec`
- 若持续低于阈值，将自动终止 pipeline 并报错
